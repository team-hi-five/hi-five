- 스크롤은 컨테이너의 크기와 콘텐츠의 크기롤 구성되어있다.

- 스크롤의 종료는 prototype으로 확인이 가능하다.

```
스크롤의 종류
  - Horizontal(가로)
  - vertical(세로)
  - Horizontal and Vertical(가로 세로 모두)
```
